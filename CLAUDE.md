This project follows strict frontend design principles. Every interface must feel hand-crafted, not AI-generated. Choose bold aesthetic directions and execute with precision. Typography must be distinctive — never use generic fonts like Inter or Roboto. Use unexpected layouts, asymmetry, overlap, generous whitespace. Add scroll-triggered animations, parallax, and micro-interactions. No cookie-cutter patterns. No template energy. Every section should feel different from the last. Commit to the aesthetic fully.
